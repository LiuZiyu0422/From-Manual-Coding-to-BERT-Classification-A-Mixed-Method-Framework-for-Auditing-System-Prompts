# From-Manual-Coding-to-BERT-Classification-A-Mixed-Method-Framework-for-Auditing-System-Prompts
This repository documents a pilot study on Claude 4.8's system prompt, tracing the methodological evolution from gold-standard annotation (50 rules, five-category typology) to semi-automated classification.本仓库记录了针对 Claude 4.8 系统提示词的试点研究，展示了从黄金标注（50条实例及五类类型学规则）到半自动化分类的方法论演进。
